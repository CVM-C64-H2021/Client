Objets structurels

Certains objets plus structurels ont également étés personnalisés, notamment les hr et les section.

Les sections sont inpirée d'une feuille de papier possée sur un bureau, carrées avec un fond blanc, projettant une ombre

Cela permet de définir facilement des sections et sous-sections distinctes.

Les hr sont également personnalisés, permettant de rapidement diviser le contenu d'une longue section avec la couleur d'accent.

Ces hr prennent 45% de la largeur de l'élément parent et offrent des marges généreuse pour bien séparer le contenu.

Les optionnels

Quelques styles accessiblent via des classes custom, mais non obligatoire!

Classes pour changer l'allure du texte

.caption

La classe .caption permet de faire une description sous une image, un tableau, etc. Le texte est placé un peu plus près de l'élément et a une fonte différente.

Ceci est une description ou explication pour l'image.

.code

Utile pour mettre en évidence que nous parlons d'éléments provennant de code. Met la fonte du texte en Space Mono. Peut être utiliser autant sur un span dans un élément, que sur un élément au complet.

Ceci est un paragraphe sur lequel l'effet code est appliqué.

.italique

Assez self explainatory, met le texte dans l'élément avec la classe en italique.

.bold

Une autre classe assez simple, permet de mettre du texte en gras.

.font-titre

Permet d'avoir la même fonte que les titres (et bouttons) sur n'importe quel élément.

.centrer-texte

Permet de centrer le texte sur lequel la classe est appliquée.

Je suis un paragraphe avec la classe .centrer-texte

.couleur-accent

Permet d'appliquer la couleur d'accent à un élément qui serait habituellement une autre couleur.

ATTENTION! Cette classe ne peut changer la couleur que sur les éléments de type texte: p, a, h1, ...

Classes pour changer le style de la page

.feature

Permet de mettre un extrait, image ou paragraphe en évidence.

Ceci est un paragraphe mit en évidence grade à la classe .feature

La classe .feature peut également être appliquée à un div pour mettre en évidence toute une section, comme cette image avec la description.

.couleur-background

Permet de définir une couleur de fond différente du reste.

Utilise pour partager un point important, un rappel ou un avertissement.

Il est possible de définir cette couleur dans les variables du thème.

.centrer-self

Permet de centrer l'élément qui possède la classe. ATTENTION! Seulement applicable pour les éléments individuel qui ne sont pas du texte.

.centrer-content

En appliquant cette classe sur un élément parent, le contenu sera centrer horizontalement et verticallement.

Cette image et description sont dans un div ayant la classe .centrer-content

.quote

Permet de faire une citation hors du flot du texte et la centrer automatiquement dans la page.

Va avec la classe .quote-auteur pour nommer l'auteur de la citation. Doit respecter la forme suivante:

<p class='quote'> Citation ... <span class='quote-auteur'> Auteur </span></p>

The purpose of art is washing the dust of daily life off our souls. Pablo Picasso
